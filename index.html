Som van lijst met getallen berekenen

Stel dat je een lijst hebt met getallen en je wilt de som van alle getallen berekenen. Zorg ervoor dat je een functie schrijft die elke lijst van getallen bij elkaar op kan tellen. Print het reulstaat naar de console.

Maak gebruik van een for loop om elk getal in de lijst te doorlopen.

Code Runner

Installeer de Code Runner door het volgende commando in je terminal uit te voeren:

docker run -v /var/run/docker.sock:/var/run/docker.sock --rm -p 8080:8080 ghcr.io/windesheim-hbo-ict/coderunner
